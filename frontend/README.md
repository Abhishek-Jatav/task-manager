# 🎨 Task Manager Frontend

A modern, responsive, and high-performance frontend built with **Next.js App Router**, delivering a seamless task management experience.

Designed with **clean UI principles**, optimized rendering, and scalable component architecture.

## 🚀 Features

✨ Users can:

* 🔐 Register & Login securely
* 📝 Create and manage tasks
* ✅ Toggle task completion
* 🗑 Delete tasks instantly
* 🔍 Search tasks in real-time
* 🎯 Filter (Completed / Pending)
* 📄 Paginate large task lists
* 🚪 Logout securely

## 🧠 Frontend Architecture

```id="arch1"
App Router (Next.js)
      ↓
Pages & Layouts
      ↓
Reusable Components
      ↓
API Layer (Axios)
      ↓
Backend (NestJS API)
```

## 🔗 API Integration

The frontend communicates with the backend using **Axios**.

### Base URL

```id="api1"
http://localhost:3000
```

### Authentication Flow

* Token stored in `localStorage`
* Sent via:

```id="api2"
Authorization: Bearer <token>
```
