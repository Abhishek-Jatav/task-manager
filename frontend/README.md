# Task Manager Frontend

Frontend client for the Task Manager application built with **Next.js**.

---

## 🚀 Tech Stack

* Next.js (App Router)
* TypeScript
* Tailwind CSS
* Axios
* React Hot Toast

---

## ⚙️ Setup

Install dependencies:

```
npm install
```

Run development server:

```
npm run dev
```

Open browser:

```
http://localhost:3000
```

---

## 📂 Pages

```
/register     User registration
/login        User login
/dashboard    Task management
```

---

## ✨ Features

Users can:

* Register
* Login
* Create tasks
* Toggle task completion
* Delete tasks
* Search tasks
* Filter tasks
* Paginate tasks
* Logout

---

## 🔗 API Connection

Frontend communicates with backend using **Axios**.

Base URL:

```
http://localhost:3000
```

Token is stored in:

```
localStorage
```

and sent via Authorization header.

---

## 🎨 UI

Styled with **Tailwind CSS** for responsive design.

Works on:

* Desktop
* Mobile
